# Task Master - Gerenciador de Tarefas com Autenticação

**Task Master** é um gerenciador de tarefas moderno desenvolvido com **HTML, CSS e JavaScript**, que permite aos usuários criar contas, fazer login e organizar tarefas com diferentes níveis de prioridade.

O projeto utiliza **LocalStorage para persistência de dados**, permitindo que usuários salvem suas tarefas diretamente no navegador.

---

## 🚀 Funcionalidades

### 🔐 Sistema de Autenticação
- Cadastro de usuários
- Login com email e senha
- Logout
- Validação de formulários

---

### 📋 Gerenciamento de Tarefas
- Criar novas tarefas
- Marcar tarefas como concluídas
- Excluir tarefas
- Definir prioridade:
  - 🔴 Alta
  - 🟠 Média
  - 🟢 Baixa
